# CLI Reference

Sutras provides a comprehensive CLI for skill development, testing, and distribution.

## Command Categories

### Skill Development

Create, inspect, and validate skills:

- [`sutras new`](new.md) - Create a new skill
- [`sutras list`](list.md) - List available skills
- [`sutras info`](info.md) - View skill details
- [`sutras validate`](validate.md) - Validate skill structure

### Testing & Evaluation

Test and evaluate skills:

- [`sutras test`](test.md) - Run skill tests
- [`sutras eval`](eval.md) - Evaluate with metrics

### Distribution

Package and share skills:

- [`sutras build`](build.md) - Build distributable package
- [`sutras publish`](publish.md) - Publish to registry
- [`sutras install`](install.md) - Install skills
- [`sutras uninstall`](uninstall.md) - Uninstall skills

### Registry Management

Manage skill registries:

- [`sutras registry`](registry.md) - Registry commands
