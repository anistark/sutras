# sutras info

View detailed information about a specific skill.

## Usage

```sh
sutras info <name>
```

## Arguments

| Argument | Description | Required |
|----------|-------------|----------|
| `name` | Name of the skill to inspect | Yes |

## Examples

### View skill details

```sh
sutras info my-skill
```

## Output

Displays detailed information including:

- Skill name and description
- Version
- Author
- License
- Allowed tools
- Dependencies
- Distribution metadata (tags, category)
- File locations
